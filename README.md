# Teq Shipment Docs Generator

Desktop tool for generating shipment documents and labels.

Supported options:

- AUS
- EU
- CA
- INDIGO

## Windows

Windows publish builds are created locally with `tjx_aus_eu_processor.spec`.

## Mac

Use GitHub Actions to build the Mac `.pkg` without installing Xcode on staff MacBooks.

1. Open the repository on GitHub.
2. Go to `Actions`.
3. Run `Build Mac Installer`.
4. Download the `Teq-Shipment-Docs-Generator-pkg` artifact.

See `README_GITHUB_MAC_BUILD.md` for details.
